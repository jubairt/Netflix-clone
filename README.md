# Project Overview

This project recreates the Netflix homepage interface using pure HTML and CSS.

The webpage includes:

- Hero section
- Navigation bar
- Sign-in button
- Email signup form
- Feature showcase sections
- FAQ section
- Footer links
- Responsive Netflix-inspired UI
